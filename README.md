### Roadmap: Система автоматического протоколирования совещаний

**Lab 1: Инициация и планирование**
*   Разработать подробный план проекта на семестр.

**Lab 2: Исследование технологий и выбор стека**
*   Изучить библиотеки для локального распознавания речи (STT): OpenAI Whisper, Faster-Whisper, Vosk.
*   Изучить языковые модели (LLM) для суммаризации текста.
*   Определиться с инструментами разработки (Python, Docker, PyTorch).

**Lab 3: Разработка модуля распознавания (STT)**
*   Настройка окружения для работы с GPU/CPU.
*   Реализация захвата и обработки звукового потока.
*   Базовая настройка модели распознавания: преобразование тестового аудио в "сырой" текст.
*   Внедрение диаризации (разделение текста по ролям: Спикер 1, Спикер 2).

**Lab 4: Разработка модуля структурирования (NLP)**
*   Разработка промптов (инструкций) для LLM по превращению текста в протокол.
*   Реализация алгоритма выделения ключевых тем, принятых решений и списка поручений.
*   Первое тестирование связки "Текст -> Черновик протокола" (Реализация 30-50%).

**Lab 5: Интеграция системы и доработка**
*   Объединение модулей STT и LLM в единый конвейер (pipeline).
*   Формирование финального вывода в читаемый формат (Markdown, DOCX или PDF).
*   Завершение разработки логики протоколирования.

**Lab 6: Демонстрация и оформление**
*   Проведение финального тестирования на записи реального внутреннего совещания.
*   Замер метрик точности распознавания и корректности выделения задач.
*   Подготовка презентации и видео.
*   Оформление технической документации и отчета.
